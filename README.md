## ğŸš€ Setup Instructions

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- Azure OpenAI resource with a GPT-4 deployment

### Backend Setup

1. **Navigate to the backend folder:**
   ```powershell
   cd backend
   ```

2. **Install dependencies:**
   ```powershell
   npm install
   ```

3. **Configure environment variables:**
   
   The `.env` file is already configured with your credentials:
   ```
   AZURE_OPENAI_ENDPOINT=https://open-ai-blusense.openai.azure.com
   AZURE_OPENAI_API_KEY=290bb1ae36cd437195b15f1a2938b52b
   AZURE_OPENAI_DEPLOYMENT_NAME=GPT-41
   AZURE_OPENAI_API_VERSION=2025-01-01-preview
   PORT=3001
   ```

4. **Start the backend server:**
   ```powershell
   npm run dev
   ```
   
   The backend will run on `http://localhost:3001`

### Frontend Setup

1. **Open a new terminal and navigate to the frontend folder:**
   ```powershell
   cd frontend
   ```

2. **Install dependencies:**
   ```powershell
   npm install
   ```

3. **Start the development server:**
   ```powershell
   npm run dev
   ```
   
   The frontend will run on `http://localhost:5173`

4. **Open your browser:**
   
   Navigate to `http://localhost:5173` to start playing!

## ğŸ”§ API Endpoints

### Backend API

- **GET** `/api/game/themes` - Get available themes
- **POST** `/api/game/start` - Start a new game
  ```json
  {
    "theme": "medieval-fantasy",
    "characterName": "Adventurer"
  }
  ```
- **POST** `/api/game/action` - Make a choice
  ```json
  {
    "sessionId": "game_xxx",
    "action": "Enter the mysterious cave"
  }
  ```
- **GET** `/api/game/state/:sessionId` - Get current game state
- **GET** `/health` - Health check endpoint

## ğŸ› Troubleshooting

### Backend won't start
- Check that your Azure OpenAI credentials are correct in `.env`
- Ensure port 3001 is not already in use
- Verify Node.js version is 18+

### Frontend can't connect to backend
- Confirm backend is running on port 3001
- Check `.env` in frontend has correct API URL
- Verify CORS is enabled in backend

### AI responses are slow
- This is normal - Azure OpenAI may take 5-10 seconds
- Consider reducing `max_tokens` in `openai.js` for faster responses
- Check your Azure OpenAI quota and rate limits
